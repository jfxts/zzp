碎片化时代的阅读

最近发现自己越来越难以读完一本长篇小说。信息的获取变成了流式（Stream）的，而非块状（Block）的。我们像是在河边用筛子淘金，留下的只有只言片语。

试图重新建立“深度阅读”的习惯，从强制自己每天关闭网络一小时开始。

磨砂玻璃效果的参数调整

为了达到完美的 Glassmorphism 效果，背景的透明度和模糊度需要特定的比例。记录一下当前网站用的参数：

.glass { background: rgba(255, 255, 255, 0.03); backdrop-filter: blur(10px); border: 1px solid rgba(255, 255, 255, 0.05); box-shadow: 0 4px 30px rgba(0, 0, 0, 0.1); }

关键在于边框的透明度要比背景稍微高一点点，这样才能有“厚度”感。

2024.03.15 #Idea

关于“未应之境”的命名

为什么叫未应之境？因为在计算机网络中，Pending (待定) 是一种美妙的状态。

"It's not failed, it's just not answered yet."

这种状态充满了可能性，就像薛定谔的猫。在请求被响应之前，一切皆有可能。
